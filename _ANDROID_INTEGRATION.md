# Kive Apps — Android Integration Guide

> **Private document.** Excluded from GitHub Pages output.

This is the reference guide for any Kive Apps Android application that consumes the encrypted API system. Hand this file to your AI agent when implementing or maintaining the SDK in an Android codebase.

## Audience

- Android engineers (or AI agents writing Kotlin)
- Reviewers verifying the implementation matches the protocol
- Future maintainers

## Prerequisites

- Kotlin 1.9+
- minSdk 24 (for AES-GCM via `Cipher` with `AES/GCM/NoPadding`)
- HTTP client (OkHttp recommended; the protocol is HTTP-only — no special headers required)
- The hardcoded **per-version key material** (see §1)

## 1. Hardcoded Per-Version Material

Each APK build hardcodes constants derived from the master secret + this APK's `version_code`. **The master secret never ships in the APK** — only the derived keys for this version's endpoints.

```kotlin
// Auto-generated at APK build time from kiveapps-source's master secret.
// One block per app version. Older versions can be removed when min_supported_code rises.
object KiveApiKeys {
    const val APP_SLUG = "lifemaster-ai"
    const val VERSION_CODE = 1
    const val BASE_URL = "https://kiveapps.github.io"

    // 32-byte hex strings — generated by an offline tool from MASTER_SECRET
    // (run on the build machine; output never committed)
    const val MANIFEST_KEY_HEX = "REPLACE_AT_BUILD_TIME"
    const val MANIFEST_PATH    = "/m/REPLACE_AT_BUILD_TIME.bin"

    // Per-endpoint content keys (32 bytes hex each)
    val CONTENT_KEYS: Map<String, String> = mapOf(
        "config"        to "REPLACE_AT_BUILD_TIME",
        "announcements" to "REPLACE_AT_BUILD_TIME",
        "notifications" to "REPLACE_AT_BUILD_TIME",
        "popups"        to "REPLACE_AT_BUILD_TIME",
        "blockers"      to "REPLACE_AT_BUILD_TIME",
        "features"      to "REPLACE_AT_BUILD_TIME",
        "premium"       to "REPLACE_AT_BUILD_TIME",
        "segments"      to "REPLACE_AT_BUILD_TIME",
        "experiments"   to "REPLACE_AT_BUILD_TIME",
        "content"       to "REPLACE_AT_BUILD_TIME",
        "emergency"     to "REPLACE_AT_BUILD_TIME"
    )

    // Stable endpoint directory hashes (per app+version+endpoint)
    val ENDPOINT_DIRS: Map<String, String> = mapOf(
        "config"        to "REPLACE_AT_BUILD_TIME",
        // ... etc.
    )

    // 32-byte HMAC key for hashing user_id (HMAC(USER_HASH_KEY, install_id||android_id))
    const val USER_HASH_KEY_HEX = "REPLACE_AT_BUILD_TIME"
}
```

### How keys are generated

A small utility script (kept in `kiveapps-source/tools/derive-apk-keys.js` — to be added when needed) takes the master secret + a version_code + an app_slug and emits a Kotlin file with all derived keys filled in. Run this once per APK release on the build machine, paste the output into the project, build the APK, then **delete the generated Kotlin file from your local disk**. The keys live only inside the signed APK.

This is a deliberate tradeoff: the keys are in the APK (an attacker can extract them by decompiling). We accept this because per-version rotation limits damage. See `_ARCHITECTURE.md` §1 ("What we explicitly do NOT protect against").

## 2. File Format Decoder

```kotlin
import javax.crypto.Cipher
import javax.crypto.spec.GCMParameterSpec
import javax.crypto.spec.SecretKeySpec
import java.security.MessageDigest

object KapFile {
    private val MAGIC = byteArrayOf(0x4B, 0x41, 0x50, 0x00)
    private const val VERSION = 0x01.toByte()

    /**
     * Decrypt a `.bin` payload produced by build.js.
     *
     * @throws SecurityException if magic, version, or auth tag is invalid.
     */
    fun decrypt(
        bytes: ByteArray,
        keyHex: String,
        appSlug: String,
        versionCode: Int,
        endpoint: String
    ): ByteArray {
        require(bytes.size >= 33) { "kap: file too small" }
        require(bytes.copyOfRange(0, 4).contentEquals(MAGIC)) { "kap: bad magic" }
        require(bytes[4] == VERSION) { "kap: unsupported version ${bytes[4]}" }

        val iv = bytes.copyOfRange(5, 17)        // 12 bytes
        val tag = bytes.copyOfRange(17, 33)      // 16 bytes
        val ct = bytes.copyOfRange(33, bytes.size)

        // AES-GCM in JCE expects ciphertext || tag concatenated.
        val ctPlusTag = ct + tag

        val key = SecretKeySpec(keyHex.hexToBytes(), "AES")
        val aad = "$appSlug|$versionCode|$endpoint".toByteArray(Charsets.UTF_8)

        val cipher = Cipher.getInstance("AES/GCM/NoPadding")
        cipher.init(Cipher.DECRYPT_MODE, key, GCMParameterSpec(128, iv))
        cipher.updateAAD(aad)
        return cipher.doFinal(ctPlusTag)
    }
}

private fun String.hexToBytes(): ByteArray =
    chunked(2).map { it.toInt(16).toByte() }.toByteArray()
```

## 3. Envelope Validation

```kotlin
import org.json.JSONObject

data class Envelope(
    val v: Int,
    val t: Long,
    val ttl: Long,
    val app: String,
    val vc: Int,
    val ep: String,
    val data: JSONObject
) {
    val isFresh: Boolean
        get() = (System.currentTimeMillis() - t) < ttl * 1000

    fun matches(expectedApp: String, expectedVc: Int, expectedEp: String): Boolean =
        app == expectedApp && vc == expectedVc && ep == expectedEp
}

object EnvelopeParser {
    fun parse(plaintext: ByteArray): Envelope {
        val json = JSONObject(plaintext.toString(Charsets.UTF_8))
        require(json.getInt("v") == 1) { "envelope: unsupported v" }
        return Envelope(
            v = json.getInt("v"),
            t = json.getLong("t"),
            ttl = json.getLong("ttl"),
            app = json.getString("app"),
            vc = json.getInt("vc"),
            ep = json.getString("ep"),
            data = json.getJSONObject("d")
        )
    }
}
```

**Always verify** `envelope.matches(...)` and `envelope.isFresh` before trusting `envelope.data`.

## 4. Manifest Fetch + Decryption

```kotlin
class KiveManifestRepo(
    private val http: OkHttpClient,
    private val cache: SharedPreferences
) {
    private val CACHE_KEY = "kive_manifest_cached"
    private val CACHE_TS  = "kive_manifest_ts"

    suspend fun getEndpointPaths(): Map<String, String> = withContext(Dispatchers.IO) {
        val cachedTs = cache.getLong(CACHE_TS, 0)
        val ageMin = (System.currentTimeMillis() - cachedTs) / 60_000
        val cachedJson = cache.getString(CACHE_KEY, null)

        if (cachedJson != null && ageMin < 60) {
            return@withContext parsePaths(JSONObject(cachedJson))
        }

        val url = KiveApiKeys.BASE_URL + KiveApiKeys.MANIFEST_PATH
        val resp = http.newCall(Request.Builder().url(url).build()).execute()
        val bytes = resp.body!!.bytes()

        val plaintext = KapFile.decrypt(
            bytes,
            KiveApiKeys.MANIFEST_KEY_HEX,
            KiveApiKeys.APP_SLUG,
            KiveApiKeys.VERSION_CODE,
            "_manifest"
        )
        val env = EnvelopeParser.parse(plaintext)
        require(env.matches(KiveApiKeys.APP_SLUG, KiveApiKeys.VERSION_CODE, "_manifest"))
        require(env.isFresh) { "manifest stale" }

        cache.edit()
            .putString(CACHE_KEY, env.data.toString())
            .putLong(CACHE_TS, System.currentTimeMillis())
            .apply()

        parsePaths(env.data)
    }

    private fun parsePaths(d: JSONObject): Map<String, String> {
        val endpoints = d.getJSONObject("endpoints")
        val out = mutableMapOf<String, String>()
        endpoints.keys().forEach { ep ->
            out[ep] = endpoints.getJSONObject(ep).getString("path")
        }
        return out
    }
}
```

## 5. Endpoint Fetch

```kotlin
class KiveEndpointRepo(
    private val http: OkHttpClient,
    private val manifestRepo: KiveManifestRepo
) {
    suspend fun fetch(endpoint: String): JSONObject = withContext(Dispatchers.IO) {
        val paths = manifestRepo.getEndpointPaths()
        val path = paths[endpoint]
            ?: throw IllegalStateException("Endpoint '$endpoint' not in manifest")

        val url = KiveApiKeys.BASE_URL + path
        val resp = http.newCall(Request.Builder().url(url).build()).execute()
        val bytes = resp.body!!.bytes()

        val keyHex = KiveApiKeys.CONTENT_KEYS[endpoint]
            ?: throw IllegalStateException("No key for '$endpoint'")

        val plaintext = KapFile.decrypt(
            bytes, keyHex, KiveApiKeys.APP_SLUG, KiveApiKeys.VERSION_CODE, endpoint
        )
        val env = EnvelopeParser.parse(plaintext)
        require(env.matches(KiveApiKeys.APP_SLUG, KiveApiKeys.VERSION_CODE, endpoint))
        require(env.isFresh) { "$endpoint stale" }

        env.data
    }
}
```

## 6. User-ID Hashing (for Premium Lookups)

```kotlin
import javax.crypto.Mac

object KiveUserId {
    @Volatile private var cached: String? = null

    fun get(context: Context): String {
        cached?.let { return it }
        synchronized(this) {
            cached?.let { return it }

            val installId = AppInstallation.getOrCreateId(context) // your stable id
            val androidId = Settings.Secure.getString(
                context.contentResolver, Settings.Secure.ANDROID_ID
            ) ?: ""

            val key = SecretKeySpec(KiveApiKeys.USER_HASH_KEY_HEX.hexToBytes(), "HmacSHA256")
            val mac = Mac.getInstance("HmacSHA256").apply { init(key) }
            val out = mac.doFinal("$installId|$androidId".toByteArray(Charsets.UTF_8))
            cached = out.toHex()
            return cached!!
        }
    }
}

private fun ByteArray.toHex() = joinToString("") { "%02x".format(it) }
```

Use the **first 8 hex chars** to look up `premium.user_grants[<prefix>]`, then verify the full hash matches if the value is an array (collision case).

## 7. Refresh Strategy

```kotlin
class KiveRefreshScheduler(
    private val app: Application,
    private val endpointRepo: KiveEndpointRepo,
    private val configStore: ConfigStore
) {
    fun init() {
        // Run on cold boot
        CoroutineScope(Dispatchers.IO).launch {
            refreshAll()
        }
        // Schedule periodic via WorkManager
        WorkManager.getInstance(app).enqueueUniquePeriodicWork(
            "kive-refresh",
            ExistingPeriodicWorkPolicy.KEEP,
            PeriodicWorkRequestBuilder<KiveRefreshWorker>(180, TimeUnit.MINUTES)
                .setConstraints(
                    Constraints.Builder()
                        .setRequiredNetworkType(NetworkType.CONNECTED)
                        .build()
                )
                .build()
        )
    }

    suspend fun refreshAll() {
        // 1. emergency first — abort if kill switch is on
        val emergency = endpointRepo.fetch("emergency")
        if (emergency.getJSONObject("kill_switch").getBoolean("enabled")) {
            handleKillSwitch(emergency.getJSONObject("kill_switch"))
            return
        }

        // 2. config (refresh interval may change)
        val config = endpointRepo.fetch("config")
        configStore.save(config)

        // 3. blockers + features + premium (parallel)
        coroutineScope {
            launch { configStore.save("blockers", endpointRepo.fetch("blockers")) }
            launch { configStore.save("features", endpointRepo.fetch("features")) }
            launch { configStore.save("premium", endpointRepo.fetch("premium")) }
            launch { configStore.save("segments", endpointRepo.fetch("segments")) }
        }

        // 4. lower-priority (lazy / on-demand)
        coroutineScope {
            launch { configStore.save("announcements", endpointRepo.fetch("announcements")) }
            launch { configStore.save("popups", endpointRepo.fetch("popups")) }
            launch { configStore.save("notifications", endpointRepo.fetch("notifications")) }
            launch { configStore.save("experiments", endpointRepo.fetch("experiments")) }
            launch { configStore.save("content", endpointRepo.fetch("content")) }
        }
    }
}
```

## 8. Caching & Offline-First

- **Persist** every successful decrypted payload to disk (encrypted at rest with `EncryptedSharedPreferences` recommended).
- **Always** fall back to cache if network fails.
- **Hardcode** sensible defaults so the app works on first launch before the network completes.
- Store the last `t` (timestamp) per endpoint and **reject responses older than the cached `t`** (replay defense beyond TTL).

## 9. Error Handling

| Error | Cause | Recovery |
|-------|-------|----------|
| `bad magic` | File isn't a KAP envelope | Likely a 404 served as HTML; retry next cycle |
| `unsupported version` | Newer file format than client | Force-update is needed; older clients should set a flag and ask user |
| `AEADBadTagException` | Auth tag mismatch | Wrong key, wrong endpoint, or tampered file. **Don't retry blindly** — log + use cache |
| `manifest stale` | Server clock skew or build hung | Retry once; if still stale, use cache |
| `Endpoint not in manifest` | Server downgraded or build error | Use cache; alert via crash reporting |

**Never crash the app on a remote-config failure.** Cache-and-continue is the default behavior.

## 10. Anti-Tamper / Bot Headers

Optional: send a custom header on every request that's hard to replicate without decompiling the APK:

```kotlin
val tamperHeader = HMAC(
    APK_SIGNING_FINGERPRINT_BYTES,
    "$VERSION_CODE|${System.currentTimeMillis() / 60_000}"
).take(8).toHex()

request.newBuilder()
    .header("X-Build-Sig", tamperHeader)
    .build()
```

GitHub Pages doesn't inspect headers, so this is just defense-in-depth for any future proxy you might run. Skip if not needed.

## 11. Putting It All Together

A minimal `Application.onCreate()`:

```kotlin
class LifeMasterApp : Application() {
    lateinit var refresher: KiveRefreshScheduler
    override fun onCreate() {
        super.onCreate()
        val http = OkHttpClient.Builder().build()
        val cache = getSharedPreferences("kive", MODE_PRIVATE)
        val manifestRepo = KiveManifestRepo(http, cache)
        val endpointRepo = KiveEndpointRepo(http, manifestRepo)
        val configStore = ConfigStore(this)
        refresher = KiveRefreshScheduler(this, endpointRepo, configStore)
        refresher.init()
    }
}
```

## 12. Testing Checklist

Before shipping a new APK version:

- [ ] Decryption round-trip test passes (hardcoded test vector matches `build.js selfTest`)
- [ ] Manifest fetch + decrypt works in production environment
- [ ] All 11 endpoints decrypt successfully
- [ ] AAD mismatch is rejected (try decrypting `popups.bin` with `config` endpoint name → expect failure)
- [ ] Stale envelope (`t` > TTL ago) is rejected
- [ ] Replay attack (older `t` than last seen) is rejected
- [ ] Cache fallback works on simulated network failure
- [ ] Force-update flow triggers when `version.min_supported_code > VERSION_CODE`
- [ ] Kill switch triggers app block
- [ ] Premium grant by user_id hash works
- [ ] Feature flag rollout percentage is honored
- [ ] WorkManager periodic refresh fires at expected interval

## 13. New Version Release Procedure

1. In `kiveapps-source`, add a new `versions[]` entry in `_meta/versions.json` (e.g., `version_code: 2`).
2. Create `_source/<app>/v<new>/` with all 11 endpoint JSONs (copy from previous version, edit as needed).
3. Push → CI builds new manifest + endpoints with version_code=2 keys.
4. Run the (offline) key-derivation tool to produce the new `KiveApiKeys` Kotlin file.
5. Bump `versionCode` in `app/build.gradle` to match.
6. Build and sign the APK.
7. Update `config.json` in `kiveapps-source` to bump `version.latest_code = 2` and (when ready to deprecate older) `min_supported_code = 2`.
8. Push the APK to Play Store.
9. After rollout, optionally remove `version_code: 1` source from `_meta/versions.json` to stop building deprecated artifacts.
